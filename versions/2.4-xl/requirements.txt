# Pinned reproducible environment for Qiskit 2.4 (XL flavor).
# Based on Qiskit-documentation's nb-tester requirements (bumped to qiskit 2.4)
# plus pylatexenc, pandas, and nbgitpuller for notebook authoring/distribution.
#
# Pins are kept on Qiskit-ecosystem packages (tightly coupled, validated as a
# set). Scientific-stack packages are unpinned and resolved by pip.
#
# Some packages are amd64-only because no aarch64 wheels exist and the
# base image has no Rust/C++ toolchain to build from source:
#   - qiskit-ibm-transpiler: every 0.15.x hard-depends on
#     qiskit_ibm_ai_local_transpiler~=0.5, whose torch chain has no
#     aarch64 wheels.
#   - gem-suite: Rust extension, sdist only on aarch64. (Also pulls
#     pymatching, which is sdist only on aarch64 and needs cmake + g++.)
# arm64 users who want these can pip install them on demand after
# adding the relevant toolchain.
